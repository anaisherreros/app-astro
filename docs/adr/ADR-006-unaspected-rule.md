# ADR-005: Regla de Planetas Inaspectados

## Estado
Accepted

## Contexto
Era necesario definir con precision cuando un planeta se considera inaspectado.

## Decision
Un planeta sigue siendo inaspectado si solo presenta conjunciones.
Para dejar de ser inaspectado necesita al menos un aspecto distinto de conjuncion.

## Motivo
- coherencia con criterio interpretativo definido para este proyecto
- evitar falsos "aspectados" por acumulacion de conjunciones

## Consecuencias
### Positivas
- regla clara y consistente
- salida mas util para interpretacion psicologica

### Negativas
- puede diferir de herramientas que cuentan la conjuncion como aspecto suficiente
