# ADR-003: Set de 7 Aspectos y Semantica de Color

## Estado
Accepted

## Contexto
Habia duda entre habilitar muchos aspectos o un conjunto limitado y legible.

## Decision
Mantener solo:
- conjunction
- sextile (blue)
- trine (blue)
- semi_sextile (green)
- quincunx (green)
- square (red)
- opposition (red)

## Motivo
- coherencia con enfoque interpretativo elegido
- simplicidad para lectura de carta
- evitar ruido tecnico en fases iniciales

## Consecuencias
### Positivas
- interpretacion mas clara
- menos complejidad de mantenimiento

### Negativas
- se omiten tecnicas avanzadas de aspectos menores adicionales
